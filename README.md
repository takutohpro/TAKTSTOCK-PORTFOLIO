# TAKTSTOCK - Creative Digital Portfolio

## 概要 (Overview)
『TAKTSTOCK』は、ブランドの世界観を形にするWebデザインスタジオのポートフォリオサイトです。
お客様のビジョンに寄り添い、記憶に残るビジュアル体験を設計し、細部までこだわり抜いたハイエンドなサイト制作を提供しています。

## 特徴 (Features)
- **洗練されたデザイン:** グレーを基調とし、ぼかし効果（グラスモーフィズム）を取り入れたハイエンドなUI。
- **リッチなアニメーション:** GSAPを活用したスクロール連動アニメーションや、テキストの出現エフェクト。
- **SEO・パフォ​​ーマンス最適化:** 必要なメタタグの完備や、シンプルで軽量なファイル構成。

## 技術スタック (Tech Stack)
- HTML
- CSS (Vanilla CSS)
- JavaScript (Vanilla + GSAP)

## ディレクトリ・ファイル構成
```text
.
├── index.html       # メインのHTMLファイル (構造とSEOメタタグ)
├── style.css        # サイト全体のスタイリング
├── main.js          # アニメーション等のスクリプト制御
├── images/          # 背景画像やファビコン (SVG)
└── README.md        # サイト概要ドキュメント (当ファイル)
```
